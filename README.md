# Socrates Rubber Duck · 思维镜

A bilingual (English/Mandarin) cognitive mirror. Press mic, think out loud.

## Start

```bash
bun install && bun dev
```

Open [http://localhost:3000](http://localhost:3000) — enter and test your API keys in the browser, then start thinking.

> Requires Node.js >= 20.9.0 and Chrome/Edge (Web Speech API). Get keys at [console.anthropic.com](https://console.anthropic.com) and [app.tavily.com](https://app.tavily.com).
